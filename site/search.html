Keyword Search

Static full-text search with Pagefind

The keyword search experience is fully client-side. After the generated HTML is indexed, researchers can search document titles, visible metadata, and extracted text without a backend.

How it works

Each generated document page includes document ID, date, release status, source URL, and extracted text. Pagefind indexes the `site/` directory after `npm run build:search` runs.

This MVP uses synthetic sample records for workflow testing. It does not publish the full PDF corpus.
