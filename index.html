About this site

Static publication workflow

This site is generated from a Python harvesting and build pipeline and deployed as static HTML to GitHub Pages. PDFs are not stored in this repository; the manifest links to the originals on the State Department FOIA site. See the project README on GitHub for build instructions and methodology.

This MVP catalogs publicly released FOIA records. PDFs are hosted by the U.S. Department of State at foia.state.gov.
